# KernelBench Dataset

The Python files under `level1/`, `level2/`, `level3/`, `level4/`, and `changelog/` in this
directory are redistributed from the [KernelBench](https://github.com/ScalingIntelligence/KernelBench)
benchmark suite.

## Source

- Repository: https://github.com/ScalingIntelligence/KernelBench
- Hugging Face dataset: https://huggingface.co/datasets/ScalingIntelligence/KernelBench
- Paper: https://arxiv.org/abs/2502.10517
